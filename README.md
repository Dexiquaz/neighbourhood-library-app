# Neighbourhood Library Management System

A mobile application that allows users within the same neighbourhood to lend and borrow books using a GPS-based restriction mechanism.

## Tech Stack
- Flutter
- Supabase
- PostgreSQL
- Python (ML & NLP)

## Key Features
- Book lending & borrowing
- Neighbourhood-based discovery
- Chat system
- Return reminders
- Recommendations
- Sentiment analysis
- Usage analytics
